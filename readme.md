# Meu Portfólio

Bem-vindo ao repositório do **Meu Portfólio** – um projeto moderno e responsivo que exibe meus trabalhos e habilidades como desenvolvedora Full Stack. Este projeto utiliza HTML, CSS, JavaScript e a biblioteca [Swiper](https://swiperjs.com/) para criar uma experiência interativa incrível.

## Visão Geral

O portfólio apresenta:

- **Layout Clean e Responsivo:** Design adaptável para diversos dispositivos.
- **Navegação Suave:** Scroll suave e menu mobile interativo.
- **Apresentação de Projetos:** Cards exibindo projetos com hover animado.
- **Slider de Tecnologias:** Exibe as tecnologias com um efeito de coverflow utilizando a biblioteca [Swiper](https://swiperjs.com/).
